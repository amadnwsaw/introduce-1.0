常見問題

可以改成接我的資料庫嗎？

可以。把搜尋函式改為呼叫你的 API 或 Firestore 集合即可。

語音搜尋支援哪些瀏覽器？

Chrome/Edge 基本可用（webkitSpeechRecognition）。Safari 可能受限。

趨勢圖表來源？

本頁 Demo 以瀏覽器 LocalStorage 記錄；正式版建議寫入後台再統計。
